Este proyecto consiste en la implementación de operaciones del álgebra relacional utilizando ASP.NET con Entity Framework, sobre la base de datos `BDVentas` en SQL Server.

## 🧩 Objetivo del Proyecto
Desarrollar consultas en C# mediante Entity Framework que representen operaciones básicas del álgebra relacional, tanto **unarias** como **binarias**.

## 🔍 Consultas implementadas

### ✅ Unarias:
- **Selección** (`σ`): Filtrado de registros bajo ciertas condiciones.
- **Proyección** (`π`): Selección de columnas específicas.
- **Renombramiento** (`ρ`): Cambio de nombres de atributos o relaciones.

### 🔁 Binarias:
- **Unión** (`∪`): Combinación de tuplas de dos conjuntos.
- **Diferencia** (`−`): Elementos de un conjunto que no están en el otro.
- **Producto cruzado** (`×`): Combinación de todos los pares posibles entre dos conjuntos.

## ⚙️ Tecnologías utilizadas

- **Lenguaje:** C# (.NET 6+)
- **Framework:** ASP.NET MVC
- **ORM:** Entity Framework Core
- **Base de datos:** SQL Server (`BDVentas`)
- **IDE:** Visual Studio 2022
